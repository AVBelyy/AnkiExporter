Excel to Anki Flash Cards Converter

How To Use It:
  • Please upload an Excel (.XLS or .XLSX) file with original and translated phrases
  • The column for original phrase must be called Source
  • The column for optional example of usage must be called Example
  • The column(s) for translated phrase must be called Target1, Target2, ...
    (you can have multiple translations, e.g. in multiple languages)
  • Underlining a character in the original phrase in Excel will add a stress sign in Anki
